About

Lolita is a light-weight Javascript plugin for creating a threaded commenting system on any page.

The source code can be found at here on github.
The demo can be found here on Heroku.

Dependencies

Lolita has ZERO dependencies. Simply include the CSS and Javascript files like so:

<link media='screen' rel='stylesheet' href='/path/to/lolita.css' />
<script src='/path/to/lolita.js'></script>

Markup

<div id='target'></div>

Javascript (Initialize)

<script>
  var editor = new Lolita('target', {
    height: '1000px'
  });
</script>

Properties

Attribute Description Default
Width Width of the editor (takes percentage or pixels). 100%
